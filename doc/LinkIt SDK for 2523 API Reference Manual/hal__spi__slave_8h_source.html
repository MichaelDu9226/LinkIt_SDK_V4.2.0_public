472 /********************* SPI Slave FSM *******************************
473 **cmd\status PWROFF PWRON ConfigRead(CR) ConfigWrite(CW)
474 ** Poweroff N Y Abnormal Abnormal
475 ** Poweron Y N N N
476 **ConfigRead N Y Abnormal Abnormal
477 ** Read N N Y Abnormal
478 **ConfigWrite N Y Abnormal Abnormal
479 ** Write N N Abnormal Y
480 
481 Note: N: invalid operation.
482  Y: the operation completed successfully.
483  Abnormal: an error occurred during the operation.
484 *******************************************************************/
